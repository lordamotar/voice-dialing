# Офлайн-система голосового ввода для RU/KZ (Offline Voice Typing)

[English Version](README.md)

Кроссплатформенное (Windows, Linux) десктопное приложение для **голосового ввода текста** и **распознавания речи (speech-to-text)**, которое работает **полностью офлайн** и печатает распознанную речь в любое активное текстовое поле (имитируя ввод с клавиатуры).

Этот проект создан для обеспечения **автономного голосового ввода** на основе движков **Vosk** и **faster-whisper**, поддерживает **автоматическое исправление грамматики** и пунктуации с помощью локального сервера **LanguageTool** и эмулирует нажатия клавиш.

## 🌟 Возможности

- **100% Офлайн и Приватность**: Всё **распознавание речи** происходит локально на вашем ПК. Никаких запросов по API к серверам, полной отсутствие интернет-трафика и конфиденциальность ваших данных.
- **Два движка распознавания**:
  - **Vosk**: Легковесный и быстрый, отлично подходит для обычных процессоров (CPU) и слабых офисных компьютеров.
  - **faster-whisper**: Высокоточное распознавание текста на базе CTranslate2. Поддерживает ускорение на видеокартах NVIDIA (CUDA) и CPU с оптимизацией `int8`.
- **Режим Push-to-Talk (Глобальная клавиша)**: Запись звука идет, пока вы удерживаете горячую клавишу (по умолчанию `ctrl+space`), и моментально расшифровывается при отпускании.
- **Умное исправление текста**: Локальный модуль LanguageTool автоматически расставляет запятые, точки, заглавные буквы и исправляет орфографические ошибки (требуется Java).
- **Поддержка языков**: Высокопроизводительные модели для **русского** (`ru`) и **казахского** (`kz`) языков.
- **Красивый индикатор (оверлей)**: Всплывающее графическое окошко-остров отображает статус (Запись, Обработка) и не забирает фокус у активной программы.
- **Интеграция с системным треем**: Приложение сворачивается в трей (область уведомлений рядом с часами) и работает в фоновом режиме. Правый клик по иконке позволяет быстро открыть «Настройки» или выполнить «Выход».
- **Автозагрузка**: Возможность настроить запуск приложения вместе со стартом операционной системы (скрыто в фоне для Windows) прямо из панели настроек.

## 💻 Системные требования

- **Python**: Версии от `3.9` до `3.12`.
- **Java (JRE/JDK)**: Нужна только для работы модуля грамматической коррекции (`language-tool-python`). Скачайте Java с [официального сайта Oracle](https://www.oracle.com/java/technologies/downloads/) или установите OpenJDK. Без Java приложение будет работать, но текст не будет корректироваться.
- **Ограничения Linux**:
  - Требуется графическая сессия **X11** (Wayland пока не поддерживается для глобального перехвата клавиатуры).
  - Требуются права суперпользователя (`sudo`) для работы библиотеки перехвата горячих клавиш `keyboard`.
  - Требуются пакеты `xclip` и `xsel` для эмуляции ввода (установка: `sudo apt-get install xclip xsel`).
- **NVIDIA GPU (Опционально для Whisper CUDA)**: Необходима видеокарта NVIDIA, установленные драйверы, CUDA Toolkit (11.x или 12.x) и библиотека cuDNN.

## 🚀 Быстрая установка «в один клик»

Проект использует современный и быстрый менеджер пакетов Python — **`uv`**.

### Windows
1. Дважды кликните по файлу **`install.bat`** в корневой папке проекта.
2. Скрипт сам скачает и настроит пакетный менеджер `uv` (если его нет), установит нужную версию Python, подгрузит библиотеки и скачает компактную модель Vosk.

### Linux / macOS
1. Откройте терминал и выполните:
   ```bash
   chmod +x install.sh
   ./install.sh
   ```
2. Скрипт подготовит среду запуска, скачает все необходимые зависимости и модель.

## ⚙️ Настройки и Графический интерфейс (GUI)

Вместо ручной правки конфигов вы можете настроить все параметры через **Панель настроек**:

### Запуск настроек
- **Windows**: Дважды кликните на **`run_settings.bat`**
- **Linux / macOS**: Запустите `./run_settings.sh` в терминале

Утилита предоставляет красивый темный интерфейс (Dark Mode) для настройки:
1. **Выбор движка**: Переключение между легким **Vosk** и точным **Whisper**.
2. **Режим работы**: Запуск на процессоре (CPU) или видеокарте (CUDA).
3. **Выбор микрофона**: Выпадающий список автоматически сканирует активные аудиовходы и выводит их реальные названия на русском языке.
4. **Параметры ввода**: Запись сочетания горячих клавиш (нажмите «Записать» и зажмите кнопки), интервал эмуляции ввода, включение оверлея и автоисправления грамматики.
5. **Автозагрузка**: Поставьте галочку «Запускать вместе с системой (Автозагрузка)», чтобы добавить приложение в автозапуск Windows (скрыто без всплывающего окна консоли) или Linux.

Также вы можете отредактировать файл **`config.yaml`** вручную (он создается автоматически из `config.example.yaml` при первом запуске):
```yaml
hotkey: "ctrl+space"                 # Горячая клавиша активации записи
language: "ru"                       # Язык ввода по умолчанию: ru | kz
engine: "vosk"                       # Движок: vosk | faster-whisper
device: "auto"                       # auto | cpu | cuda (только для Whisper)
vosk:
  model_path: "models/vosk-model-small-ru-0.22"
faster_whisper:
  model_size: "small"                # tiny | base | small | medium
  compute_type_cpu: "int8"           # Квантование для CPU
  compute_type_gpu: "float16"        # Квантование для GPU (CUDA)
enable_grammar_correction: true      # Коррекция текста через LanguageTool
enable_overlay: true                 # Отображение индикатора записи
input_device_index: null             # Индекс микрофона (null - авто)
typing_interval: 0.01                # Задержка эмуляции нажатий клавиш (сек)
log_level: "INFO"                    # Логирование: DEBUG | INFO | WARNING | ERROR
```

## 🏃 Запуск голосового ввода

После установки и настройки запустите систему распознавания:
- **Windows**: 
  - Для запуска в скрытом режиме (без отображения окна консоли на панели задач): дважды кликните на **`run_silent.vbs`** (рекомендуется).
  - Для запуска с отображением консоли: дважды кликните на **`run_app.bat`**
- **Linux / macOS**: Запустите `./run_app.sh`

### Как пользоваться голосовым вводом:
1. В консоли появится сообщение: `[ГОТОВО] Ожидание горячей клавиши 'ctrl+space'...`
2. Зажмите горячую клавишу (по умолчанию `Ctrl + Space`).
3. Говорите текст в микрофон.
4. Отпустите горячую клавишу. Приложение распознает вашу речь, расставит знаки препинания и напечатает текст в открытое активное окно.

## 🛠️ Решение проблем (Troubleshooting)

### 1. Ошибка Java / LanguageTool не запускается
- **Симптом**: Приложение сообщает об отсутствии Java. Распознавание работает, но знаки препинания и заглавные буквы не исправляются.
- **Решение**: Установите Java JRE/JDK (версии 8 или выше) и добавьте путь к исполняемому файлу `java` в системные переменные PATH.

### 2. CUDA недоступна
- **Симптом**: В режиме `device: auto` программа выдает лог об ошибке CUDA и переключается на CPU.
- **Решение**: Убедитесь, что у вас видеокарта NVIDIA, установлены актуальные драйверы, CUDA Toolkit (12.x/11.x) и cuDNN.

### 3. Wayland в Linux
- **Симптом**: Горячие клавиши или ввод текста не работают в Linux.
- **Решение**: Библиотеки `keyboard` и `pyautogui` требуют X11. Выберите сессию **X11** вместо Wayland на экране входа в систему Linux.
